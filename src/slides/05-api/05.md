### Interaction

Change element state via common state actions

```javascript
cy.type("abcdefg");
cy.click();
cy.dblclick();
cy.select();
cy.check() / uncheck();
cy.focus() / blur();
```

### Viewport

Control the size and position of the browser viewport

```javascript
cy.scrollTo() / scrollIntoView();
cy.viewport(x, y);
```
