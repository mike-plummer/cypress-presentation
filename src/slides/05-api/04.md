### Navigate

```javascript
cy.visit("http://www.objectpartners.com");
cy.go("forward" | "back");
cy.reload();
```

### Get an element

```javascript
cy.get("#id"); // by ID
cy.get(".cssClass"); // by CSS class
cy.get([(attr = value)]); // by attribute
```

### Find a child

```javascript
cy.get(".content").find("input");
```
