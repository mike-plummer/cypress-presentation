## Mocking

### Network Requests

```javascript
cy.server()
  .route("GET", "/users", [])
  .route("GET", "/data/**", "fixture:data.json");
```

### Stubbing

```javascript
cy.visit("http://localhost:8080", {
  onBeforeLoad(win) {
    cy.stub(win, "alert");
  }
})
  .window()
  .its("alert")
  .should("be.called");
```
