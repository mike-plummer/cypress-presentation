### Interaction

```javascript
cy.get("#id")
  .type("abcdefg") // Type text into element
  .click() // Single mouse click
  .dblclick() // Double mouse click
  .select() // Choose item from menu
  .check() // Handle checkbox
  .uncheck()
  .focus() // Focus/blur element
  .blur();
```

### Viewport

```javascript
cy.get("#id").scrollIntoView(); // Move viewport to include element
cy.scrollTo(x, y); // Move viewport to specific position
cy.viewport(x, y); // Resize viewport
```
